# フロントエンド課題

あなたは友人と旅行に行った際の費用精算を行うためのアプリケーションを作成するフロントエンドエンジニアです。DB は作らなくて良いのでセッション内で割り勘ができるアプリケーションを作成してください。詳細要件は以下の通りです。

## 必要要件 (国内旅行)

- 人を追加 (旅行参加者)
- 支払い費用の追加 (誰がいくら何に払ったのか)
- 精算 (総費用を割り勘して、誰が誰にいくら払えば良いのかを計算)

## 追加要件 (海外旅行)

- 複数の通貨で支払いを行った場合への対応
  - API は以下を推奨 (他で良いものがあればそれでも構いません)
  - https://exchangerate.host/documentation

## 使用技術スタック

- NextJS (app router)
- TypeScript
- TailwindCSS

## 評価ポイント

- ユーザーの使いやすさを意識した UI・UX であるかどうか
- 適切にコンポーネントに分られているか
- バグやレイアウト崩れがないかどうか
  - 数値のマイナスバリデーション
  - 割り算の小数点以下の処理等
